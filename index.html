Switch

Switch é uma estrutura de controle de fluxo que avalia uma expressão e executa o bloco de código correspondente ao valor da expressão. É usado para substituir várias instruções if e tornar o código mais legível e conciso.

getElementbyId

É um metodo usado para obter um elemento HTML por meio de seu ID único. Facilita a manipulação e atualização desse elemento.

Case

Case é uma estrutura de controle que permite testar um valor e executar um bloco de código específico com base no resultado.

InnerHTML

InnerHTML é uma propriedade que permite acessar e modificar o conteúdo HTML de um elemento. Pode ser usado para inserir, alterar ou recuperar markup dentro de um elemento, facilitando a manipulação dinâmica do conteúdo em páginas web.

Math.round()

Math.round() é uma função que arredonda um número para o inteiro mais próximo. Se a parte fracionária for 0.5 ou maior, ele arredonda para cima; caso contrário, arredonda para baixo.
